Panduan Pengguna

1. Pendahuluan

Selamat datang di SIMDOKPOL (Sistem Informasi Manajemen Dokumen Kepolisian). Aplikasi ini dirancang khusus untuk membantu unit pelayanan kepolisian (SPKT) dalam menerbitkan, mengelola, dan mengarsipkan Surat Keterangan Hilang (KTP, SIM, STNK, ATM, dll) secara cepat, presisi, dan aman.

2. Instalasi & Setup Awal

Saat aplikasi pertama kali dijalankan, Anda akan diarahkan ke Setup Wizard 5-Langkah:

Langkah Konfigurasi
  • 1 Pilih Database: Gunakan SQLite untuk penggunaan di satu laptop, atau MySQL/PostgreSQL untuk jaringan kantor (LAN).
  • 2 Koneksi: Masukkan kredensial database (Host, User, Password) jika memilih mode server.
  • 3 Identitas Instansi: Isi KOP surat (Kepolisian Negara, Daerah, Resor) dan nama kantor pelayanan.
  • 4 Penomoran: Tentukan format nomor surat (contoh: SKH/%03d/X/2025) dan nomor urut terakhir.
  • 5 Akun Admin: Buat akun Super Admin pertama Anda untuk mengelola sistem.
Tampilan Wizard Konfigurasi
3. Penerbitan Surat Keterangan
3.1. Input Data Pemohon

Masuk ke menu Buat Surat Baru. Isi data diri pelapor (Nama, TTL, Alamat) sesuai KTP. Sistem otomatis mengubah huruf menjadi KAPITAL untuk keseragaman.

3.2. Data Barang Hilang

Klik "Tambah Barang". Pilih jenis barang (KTP, SIM, STNK, dll). Form isian akan menyesuaikan secara otomatis (misal: STNK meminta No. Rangka & Mesin, sedangkan ATM meminta Nama Bank).

Tips: Anda dapat menambahkan lebih dari satu barang dalam satu surat keterangan.
4. Cetak & Pengarsipan
Cetak Presisi (PDF)

Klik tombol Preview atau PDF di tabel dokumen. Sistem akan menghasilkan file PDF ukuran A4 yang siap cetak dengan tata letak presisi.

Arsip Otomatis

Dokumen yang melewati batas waktu (default 15 hari) akan otomatis dipindahkan ke menu Arsip Dokumen. Status berubah dari DITERBITKAN menjadi DIARSIPKAN.

5. Fitur Administrasi (Super Admin)
Manajemen Pengguna

Tambah personel operator baru, reset password, atau non-aktifkan akun petugas yang mutasi.

Laporan Agregat PRO

Unduh rekapitulasi bulanan dalam format PDF/Excel yang berisi statistik jumlah surat, kinerja operator, dan tren barang hilang.

Template Barang PRO

Sesuaikan formulir isian barang secara dinamis. Tambah field baru atau ubah urutan tanpa coding.

Migrasi Data PRO

Pindahkan seluruh data dari Laptop (SQLite) ke Server (MySQL/Postgres) secara otomatis saat unit Anda berkembang.

HTTPS Aman

Aktifkan enkripsi SSL di menu Pengaturan untuk mengamankan data di jaringan Wi-Fi kantor.
