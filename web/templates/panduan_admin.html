Administrasi & Sistem

1. Manajemen Pengguna

  • Tambah operator/petugas baru di menu Manajemen Pengguna.
  • Peran menentukan akses (Super Admin atau Operator).
  • Akun non-aktif tetap tercatat di dokumen lama, namun tidak bisa login.

2. Master Jabatan

  • Kelola daftar jabatan agar input petugas konsisten.
  • Dropdown jabatan mengambil data dari master ini.
  • Jika ada jabatan khusus, gunakan opsi LAINNYA.

3. Log Audit

  • Aktivitas penting (tambah/ubah data, pengaturan, jabatan) tercatat di Log Audit.
  • Gunakan untuk pelacakan perubahan dan monitoring.

4. Backup, Restore, dan Migrasi

  • Backup dan Restore berlaku untuk SQLite.
  • Migrasi menyalin seluruh data ke MySQL/Postgres.
  • Pastikan DB target kosong sebelum migrasi.

5. HTTPS & Sertifikat

  • Aktifkan HTTPS di menu Pengaturan untuk jaringan kantor.
  • Gunakan fitur Install Sertifikat agar browser percaya pada sertifikat lokal.
  • Untuk akses via perangkat lain, pastikan sertifikat CA sudah terpasang.

6. Lisensi & Aktivasi

  • Menu Upgrade menampilkan status lisensi dan QR HWID.
  • Masukkan Activation Code sesuai prosedur aktivasi offline.
