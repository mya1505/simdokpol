Informasi Sistem

SIMDOKPOL

Sistem Informasi Manajemen Dokumen Kepolisian

SIMDOKPOL merupakan aplikasi desktop modern berbasis web yang dirancang untuk membantu unit kepolisian dalam manajemen dan penerbitan Surat Keterangan Hilang secara efisien, cepat, dan aman. Aplikasi ini mendukung deployment dari skala kecil (laptop tunggal) hingga skala besar (server atau cloud infrastructure).

Fitur Unggulan
Setup Wizard Multi-Database Cetak PDF Presisi Template Dinamis Audit Log Role-Based Security
Teknologi
  • Core: Golang Backend dengan Gin Framework
  • Database: GORM (ORM) & Excelize untuk Excel
  • Interface: Metro Style dengan Bootstrap 4
  • Security: HMAC Licensing, JWT Authentication, Bcrypt
2025 SIMDOKPOL. Dikembangkan untuk Unit Kepolisian Indonesia.
